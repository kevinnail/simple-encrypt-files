Commands for script:


1)	Open your terminal and navigate to this directory containing encrypt_files.js.

2)	Run the script with the folder path and password as arguments:

		node encrypt_files.js /path/to/your/folder your-password

3) 	Can be any string for password




Encrypt example:	node C:\Projects\encrypt_files.js "C:\Projects\MyJournalEntries" my-secret-password

Decrypt example:	node C:\Projects\decrypt_files.js "C:\Projects\MyJournalEntries" my-secret-password
